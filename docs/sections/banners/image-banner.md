### What is the Image Banner section?

The "Image Banner" section is a prominent visual display that showcases your brand or promotions with a blend of high-impact imagery and succinct text. Easily customizable, it sets the stage for your store's narrative and key offerings.

![](/img/docs/image-banner-section.jpeg)

### Main settings

![](/img/docs/image-banner-settings-2.jpeg)

**Heading**

- **Heading:** Set the title for the image banner to capture the audience's attention.

**Make Section Full Width**

- **Make section full width:** Enable this setting to make the image banner span the entire width of the container.

**Section Height**

- **Height:** Choose between adapting the height to the image or setting a fixed height.
- **Desktop fixed height:** Adjust the fixed height for desktop view.
- **Mobile fixed height:** Adjust the fixed height for mobile view.

**Colors**

- **Text color:** The 'Transparent' value is the default and subsequently inherits the value from the global settings.
- **Background color:** The 'Transparent' value is the default and subsequently inherits the value from the global settings.
- **Background opacity:** Adjust the opacity of the background color (default 100%).

**Large Screen Version**

- **Image:** Select an image for the large screen version of the banner.
- **Large screen text alignment:** Choose the alignment of the text on the banner for large screens (vertical: Top, Center, Bottom; horizontal: Left, Center, Right).

**Mobile Version**

- **Image:** Select an image for the mobile version of the banner.
- **Mobile screen text alignment:** Choose the alignment of the text on the banner for mobile screens (vertical: Top, Center, Bottom; horizontal: Left, Center, Right).

**Section Spacing**

- **Distance from the top:** Adjust the spacing from the top of the section.
- **Distance from the bottom:** Adjust the spacing from the bottom of the section.

### Customize Countdown Timer block

The **Countdown Timer** block is a dynamic feature designed to create urgency and encourage customer action by displaying a countdown to a specific date and time. This block is perfect for highlighting limited-time offers, upcoming events, or product launches. It helps drive engagement and motivates customers to act before the timer expires.

![](/img/docs/image-banner-countdown-block.jpeg)
