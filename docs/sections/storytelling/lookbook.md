### What is the Lookbook section?

The "Lookbook" section provides an elegant way to feature a collection of images in a clean, organized layout. This visually engaging section can highlight different aspects of a collection, editorial content, or various products in a straightforward yet impactful manner. It's designed to draw attention to specific items or themes, making it a versatile tool for storytelling and product display.

![](/img/docs/lookbook-section.jpg)

### Main settings

![](/img/docs/lookbook-settings.png)

**Heading**

- **Heading:** Set the title for the lookbook section to introduce the content.
- **Use rich text field for heading:** Enable a rich text heading to choose the heading level (H1–H6) for SEO purposes and connect dynamic sources if needed.

**Show icons**

- **Show icons:** If this setting is enabled, a plus button will appear on the image when a link is set.

**Disable on Mobile**

- **Disable on mobile:** Enable this setting to hide the lookbook section on mobile devices.

**Make Section Full Width**

- **Make section full width:** Enable this setting to make the lookbook section span the entire width of the container.

**Large Screen Version**

- **Gap:** Adjust the gap between items in the lookbook on large screens.

**Mobile Version**

- **Gap:** Adjust the gap between items in the lookbook on mobile devices.

**Colors**

- **Text color:** The 'Transparent' value is the default and subsequently inherits the value from the global settings.
- **Background color:** The 'Transparent' value is the default and subsequently inherits the value from the global settings.
- **Background opacity:** Adjust the opacity of the background color.

**Section Spacing**

- **Distance from the top:** Adjust the spacing from the top of the section.
- **Distance from the bottom:** Adjust the spacing from the bottom of the section.

### Customize image blocks

![](/img/docs/lookbook-image-block.jpg)

- **Image:** Provides the ability to choose an image for the Lookbook block from an existing library or upload a new one.
- **Style:** Offers two styling options for the image block – Primary and Secondary. This determines the visual hierarchy of the image within the Lookbook section, allowing for design flexibility and emphasis where needed.
- **Link:** Enables a link to be attached to the image, leading the user to relevant content or products when they interact with the image.
