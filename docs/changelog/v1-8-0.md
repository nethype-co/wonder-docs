# v1.8.0 — April 29, 2025

## Added

- New Navigation Layout: **Sleek header** with logo on left, centered nav links, optional transparent style
- Enhanced **Country/Language Selector**: language and country switching with built-in search in header and footer
- New **Testimonial Images** section: customer feedback display with images and quotes
- **Password Page** update: fully customizable password page for brand matching
- **Charm Icon Set**: fresh icons with custom SVG upload support

## Fixes and Improvements

- **Newsletter Popup**: delay popup display option after page load for better conversions
- **Color Scheme Improvements**: enhanced control for dark/inverted theme versions
- **Cart Drawer Enhancements**: hide cross-sell blocks, improved animations and spacing
- **Feature Icons** in product pages: consistent feature icons across multiple products
- Visual tweaks: spacing, alignment, and animation refinements throughout
- Improved integration with apps: public event listener for cart refresh and variant change publishing for third-party compatibility
