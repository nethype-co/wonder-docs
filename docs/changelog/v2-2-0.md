# v2.2.0 - January 22, 2026

This update introduces new content sections, richer visual customization options, and more flexible layouts across the theme, with a strong focus on PDP, media, and content-heavy pages.

## New Sections and Blocks

- **Video Player** section
- **Recently Viewed Products** section
- **Icon with Text** block for hero sections
- **Image with Table** preset and table block
- **UI Spacer** shared block for precise vertical spacing
- **Counter/Countdown** section

## Richer Content and Media Layouts

- Extended options for Image with Video sections (including "show both" on mobile)
- New image width options for blog posts and improved gallery behavior on PDP

## Typography, Color, and Backgrounds

- Refactored heading fields in many sections to support rich text and inline formatting
- Gradient background option for the Rich Text section, with dedicated heading and bold-text color controls

## Navigation, Header, and Footer

- Enhanced mega menu logic and optional category groups in main search
- More flexible header and footer groups (including new section support and slider styles)
- Additional customization options for password page header and footer

## Performance, Accessibility, and SEO

- WCAG Level A and ADA compliance improvements
- Various optimizations to increase Lighthouse scores

## Other

- Various minor fixes and refinements throughout the theme
