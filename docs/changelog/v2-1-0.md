# v2.1.0  -  October 15, 2025

## Added

- Refreshed **Velour Preset** with new logo banner animation
- New Section: **Logo Banner with Text** featuring transition effect
- New Section: **Marquee Images** for dynamic image scrolling
- New Section: **Image with Scrolling Text**
- New Block: **Specification Table** on product detail pages
- **PDP Video Reels** block on product detail pages
- **Benefit Hotspot** option in Shoppable Image section
- **Scroll Reveal Effect** added to footer section
- **Custom Font Settings** for typography control
- **Predictive Search**  -  advanced settings for improved customization
- **Combined Listing Improvements** for sibling products and variations
- New Page Presets for Bags, About, Lookbook, Sale, FAQ, and Product Page

## Changed

- Enhanced mobile navigation with improved touch interactions
- Updated promotional menu on mobile with Stack View layout
- Added product integration to Testimonial Images section
- Added dynamic sources and H1/H2/H3 heading level support in Headline/Separator section

## Fixes

- Unified video controls and autoplay behavior across components
- Fixed mobile swiper issues in Brands section
- Resolved Safari compatibility problems with logo animations
- Fixed product variant selection logic and related bugs
- Improved collection display layout consistency
- Updated translations and filled missing text
- Adjusted responsive breakpoints for better display
- Enhanced accessibility with ARIA labels and focus management
